Skills package repeatable workflows.

<div class="best-practice">
<strong>Mental model</strong><br>
Instructions = what your team expects.<br>
Agents = which role you activate.<br>
Skills = which workflow you execute.
</div>

## Example skill themes
- Reflection skill: capture learning from customer interactions and improve repo conventions
- Brainstorm/research skill: generate options before writing specs
- Spec-clarify skill: find ambiguities and mark <span class="needs-clarification">NEEDS CLARIFICATION</span>

<div class="exercise">
<strong>Try it now</strong>
<ol>
<li>Create <code>.github/skills/brainstorm/SKILL.md</code>.</li>
<li>Define sequential question fallback if multi-question tool is unavailable.</li>
<li>Commit your skill.</li>
</ol>
</div>
