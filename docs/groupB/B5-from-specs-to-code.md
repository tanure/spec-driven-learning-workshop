# B5: From Spec to Working Code

This is the payoff: your code now follows a clear requirement contract.

<div class="best-practice">
<strong>Prompting pattern</strong><br>
"Implement Feature#101. Follow the acceptance criteria and edge cases from the issue."
</div>

<div class="exercise">
<ol>
<li>Copy one Feature spec from your issues.</li>
<li>Ask Copilot to implement it in your starter repo.</li>
<li>Review output against acceptance criteria.</li>
</ol>
</div>
