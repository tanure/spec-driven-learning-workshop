# C1: The Complete Workflow

This workflow combines instructions, agents, skills, and spec-kit.

```text
Day 0: Foundation
  - .instructions.md defines stack and standards
  - agents specialize roles (reviewer, generator, docs)
  - skills automate workflows (brainstorm, clarify, reflection)

Day 1: Planning
  - Constitution issue created
  - speckit spec creates Epic/Architecture
  - clarify skill resolves NEEDS CLARIFICATION
  - speckit plan creates features

Day 1+: Delivery
  - analyze each feature for risks and complexity
  - speckit tasks creates task-level DoD
  - speckit start begins implementation on branch
  - PR review uses reviewer agent
  - merge and wiki-sync
```
