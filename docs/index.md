## How to Use This Workshop

> Choose **Group A** or **Group B** first. Complete lessons in order inside that group. Group C unlocks after you complete all lessons in Group A or Group B.

## 1-Day Hackathon Flow (90-minute training block)

| Time | Focus |
|---|---|
| 0–30 min | Group A or Group B start |
| 30–65 min | Complete remaining lessons in chosen group |
| 65–75 min | Group C integration |
| 75–90 min | Use case selection and kickoff prep |

## Outcomes

- Build reliable Copilot context with `.instructions.md`
- Create role-focused agents and workflow skills
- Run complete spec-kit lifecycle
- Avoid vibing coding through specification discipline
- Start hackathon implementation with confidence
